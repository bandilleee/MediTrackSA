# MediTrack SA 🏥

MediTrack SA is a comprehensive chronic disease management platform designed to revolutionize healthcare delivery in South Africa. By connecting patients, clinics, and community health workers, we enable better treatment outcomes through intelligent tracking, smart notifications, and AI-powered support.

## 🌟 Features

- **Smart Notifications**: Real-time alerts for medication availability and pickup schedules
- **Secure & Compliant**: POPIA-compliant data protection with advanced encryption
- **Adherence Tracking**: Comprehensive monitoring to improve treatment outcomes
- **AI-Powered Support**: Intelligent chatbot for instant guidance and health tips
- **Delivery Tracking**: Seamless coordination between clinics and community health workers
- **Interactive 3D Visualizations**: Immersive Three.js scenes for enhanced user engagement

## 🚀 Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion & Motion
- **3D Graphics**: Three.js with React Three Fiber
- **Particles**: tsParticles for dynamic backgrounds
- **UI Components**: Radix UI primitives
- **Icons**: Lucide React
- **Utilities**: clsx, tailwind-merge, class-variance-authority
